## 2026/3/3
今天学了struct结构体，但是代码实践的时候有点问题，但是有一点理解的，其实就是数组，但是可以包含和使用的范围更广，就好像python里的哈希表，但是和哈希不同的是，哈希表是通过键值对来存储数据的，而结构体是通过成员变量来存储数据的，而且结构体可以包含不同类型的成员变量，而哈希表只能包含键值对。
现在的问题是这个结构体和一般的数字可不可以进行加减乘除等操作？显然说在课件里结构体之间的是可以的，但是结构体和一般的数字是不可以进行加减乘除等操作的，因为结构体是一种自定义的数据类型，而一般的数字是一种基本的数据类型。

## 2026/3/4
今天学了指针，这其实是一种关于内存的一种数量表示方式，它是把十六进制的内存转换为一个变量（形如*ptr），而这个变量就是指针变量，它可以指向内存中的某个地址，从而可以通过指针变量来访问内存中的数据。
指针虽然是地址的一个方法，但是它是否在引用数据的时候，是直接引用数据，还是引用数据的地址，这是一个问题。明天在实践课的时候再试一下是否会改变其用时和占用内存。
